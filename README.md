# mlProj
MPEG-7 machine learning experiments  
You will need to replace the paths in every file if you want to use it on your system  
To run vde.exe and vdm.exe please download them at http://image.ntua.gr/iva/tools/vde
  
List of required libraries:  
xml  
numpy  
re  
torch  
torchvision  
PIL  
pathlib  
sklearn  
copy  
lightgbm  
  
Optional for baseline:  
os  
time  
math  
collections  
subprocess
